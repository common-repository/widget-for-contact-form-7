== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/widget-for-contact-form-7` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Appearance -> Widgets -> Widget for Contact form 7
